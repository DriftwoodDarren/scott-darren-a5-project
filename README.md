# w25-a5-workalong

Darren Scott Workalong Mod 

Changes made for the mod 
- Changed player ship to pirate ship sprite
- Added pause button
- Added Background Music
- Added Cannon Firing Noise

Controls 
- A: rotate ship left
- D: rotate ship right
- Space: Add thrust to ship (forward)
- Esc: Pause/Unpause game
